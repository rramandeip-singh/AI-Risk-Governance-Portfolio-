# Netflix Data Analysis (AWS SageMaker)

## Overview
This project analyzes Netflix dataset using AWS SageMaker to identify trends and user preferences.

## Tools Used
- Python
- AWS SageMaker
- Pandas, Matplotlib

## Key Insights
- Identified popular genres
- Analyzed content distribution
- Visualized trends over time

## Outcome
Improved understanding of large-scale data analysis using cloud-based tools.
